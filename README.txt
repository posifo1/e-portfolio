# MSc Cybersecurity ePortfolio (Template)

Includes:
- Main Page (index.html)
- Security & Risk Management Module
- Two placeholder modules for future use
- Shared style sheet (assets/style.css)

## How to Use
1. Open `index.html` in your browser to view locally.
2. Edit text in each HTML file to add your own reflections and artefacts.
3. Place PDFs, images, or reports in `assets/`.
4. Add new modules by copying a placeholder and updating links.
